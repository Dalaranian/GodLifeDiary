## 갓생 일지<br>
좋은 습관을 혼자 꾸준히 지속하기 힘든 사람들을 위한 웹 서비스!<br> 여러 사람들과 함께 장,단기 목표를 잡고 함께 달려나가는 서비스로, <br>동기부여 및 정보 공유를 통해 지루하고 반복적인 일과를 포기하지 않고 끝까지 즐길 수 있도록 도와주는 것이 목표입니다.
